Linguagens e softwares

Html

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web.

CSS

O Cascading Style Sheets (CSS) é uma "folha de estilo" composta por “camadas” e utilizada para definir a aparência em páginas da internet.

PHP

É uma linguagem de script open source de uso geral, muito utilizada, e adequada para o desenvolvimento web e que pode ser embutida dentro do HTML.

JS
